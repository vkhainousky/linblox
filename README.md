# linblox
Launch roblox on linux systems!
# Before you play :)
## Please install:
Wine [download](https://www.winehq.org)

Python [download](https://www.python.org)

Also install these python modules:

```
sudo apt-get install python3-tk
```

```
pip install customtkinter
```

```
pip install pillow
```

```
pip install pygame
```
